# Algiter

This is a little toy to loop through all interesting simple algebraic expressions and, if they are equal, print them out.

Trivial things like multiplication by one or addition of zero are excluded.

## Usage

```
python3 algiter.py
```

## Sample output

```
((1)+(x))-(1) = ((1)+(x))+(-1)
((1)+(x))-(-1) = (2)+(x)
(2)+(x) = ((1)+(x))-(-1)
((1)+(x))-(-1) = (x)+(2)
(x)+(2) = ((1)+(x))-(-1)
((1)+(x))-(-1) = (x)-(-2)
(x)-(-2) = ((1)+(x))-(-1)
((1)+(x))-(-1) = ((1)+(x))+(1)
((1)+(x))+(1) = ((1)+(x))-(-1)
((1)+(x))-(-1) = (1)+((1)+(x))
(1)+((1)+(x)) = ((1)+(x))-(-1)
((1)+(x))*(-1) = (-1)-(x)
(-1)-(x) = ((1)+(x))*(-1)
((1)+(x))/(-1) = (-1)-(x)
(-1)-(x) = ((1)+(x))/(-1)
((1)+(x))/(-1) = ((1)+(x))*(-1)
((1)+(x))*(-1) = ((1)+(x))/(-1)
(-1)+((1)+(x)) = x
x = (-1)+((1)+(x))
(-1)+((1)+(x)) = ((x)+(x))-(x)
((x)+(x))-(x) = (-1)+((1)+(x))
(-1)+((1)+(x)) = ((x)+(x))/(2)
((x)+(x))/(2) = (-1)+((1)+(x))
(-1)+((1)+(x)) = ((x)*(x))/(x)
((x)*(x))/(x) = (-1)+((1)+(x))
(-1)+((1)+(x)) = ((1)+(x))-(1)
((1)+(x))-(1) = (-1)+((1)+(x))
(-1)+((1)+(x)) = ((1)+(x))+(-1)
```
